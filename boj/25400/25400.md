# Logic
- 정렬되어있으면 제자리 상태
- 1부터 x까지(current_seq) 배열을 순서대로 돌면서 해당 값이라면 패스
  - 아니라면 skip++
- skip 출력

# My Code

## cpp

```cpp title="boj/25400.cpp"

#include <vector>
void solution()
{
    int n;
    cin >> n;

    vector<int> v(n);

    for (int i = 0; i < n; ++i) {
        cin >> v[i];
    }

    int skip = 0;
    int current_seq = 1;

    for (const auto& i : v) {
        if (i == current_seq) {
            current_seq++;
            continue;
        }
        skip++;
    }

    cout << skip;
}
```
