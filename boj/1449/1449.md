### *설계*
1. 알고리즘 선택
	- 앞에서 부터 길이만큼 true
		- 사용횟수 +1
	- true면 넘기기
2. 자료구조 선택
	- 벡터
3. 수도 코드 작성
4. 정합판단
  - `1 ~ 3`과정으로 도출된 로직이 문제를 해결하는가
    - 그렇다 -> **구현**
    - 잘 모르겠다 -> **구현**
    - 아니다 -> 1번부터 다시 점검

int main() {
    int n, l;
    cin >> n >> l;
    int pipe_size = 1001;

    vector<bool> pipe(pipe_size, true);
    int          count = 0;

    for (int i = 0; i < n; ++i) {
        int temp;
        cin >> temp;

        pipe[temp] = false;
    }

    for (int i = 0; i < pipe_size; ++i) {
        if (pipe[i] == false) {
            count++;
            for (int j = 0; j < l; ++j) {
                pipe[i + j] = true;
            }
        }
    }
    cout << count;
    return 0;
}
